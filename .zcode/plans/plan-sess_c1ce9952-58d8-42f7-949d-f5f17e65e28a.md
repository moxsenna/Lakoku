### File BARU (7 file)

**1. `supabase/migrations/20260711000000_reader_taste_profiles.sql`**
- Table: `id uuid PK default gen_random_uuid()`, `user_id uuid FK on delete cascade`, `taste_json jsonb not null default '{}'`, `created_at`, `updated_at`
- Unique index: `reader_taste_profiles_user_uidx` on `(user_id)`
- RLS: `select_self`, `insert_self`, `update_self` — all `to authenticated`, `auth.uid() = user_id`

**2. `lib/api/taste-profile.ts`**
- `getTasteProfileForUser(userId: string): Promise<TasteProfile | null>` — SELECT from DB via server client
- `saveTasteProfileForUser(userId: string, profile: TasteProfile): Promise<void>` — upsert `{ user_id, taste_json, updated_at }` onConflict `user_id`
- Guest early-return di caller (server action), bukan di sini

**3. `app/onboarding/selera/actions.ts`**
- `actSaveTasteProfile(rawInput)` — validate schema, dynamic import `getSessionUser`, `saveTasteProfileForUser(user.id, profile)`, return `{ ok: true }`
- `actGetTasteProfile()` — return server profile atau `null`
- `actSkipTasteProfile()` — save profile kosong + `skippedAt` + `updatedAt`
- `actMergeGuestTasteProfile(guestProfile)` — merge: server kosong → pakai guest; server ada → server menang; tidak overwrite diam-diam

**4. `app/onboarding/selera/page.tsx`**
- Server component, `force-dynamic`
- Import `TasteProfileFlow` langsung (no dynamic import wrapper)
- Metadata: "Atur Selera Cerita — Lakoku"

**5. `app/onboarding/selera/loading.tsx`**
- Skeleton loading screen mengikuti pola `MulaiSkeleton` di `app/mulai/page.tsx`

**6. `components/onboarding/taste-profile-flow.tsx`**
- Client component, 5 langkah quiz (genre → trope disukai → dihindari/batas → intensitas → gaya/ending)
- State: step, answers partial Profile, `next` from searchParams
- Max 6 opsi visual per langkah, multi-select dengan microcopy "Pilih satu atau beberapa"
- Submit: `actSaveTasteProfile` → `router.push(next || '/beranda')`
- Skip: `actSkipTasteProfile` → `router.push(next || '/beranda')`
- Styling: ikuti existing pattern (rounded-2xl, font-serif, primary CTA)

**7. `scripts/taste-profile-db-smoke.ts`**
- Static check: migration file exists, table/index/RLS statements ada di SQL, upsert onConflict pattern di `lib/api/taste-profile.ts`, merge logic di schema
